# Weather search application
This application is a pet project. This project implements the weather display for the city specified in the search field. The Weather API was used to display the weather

## Stack
- HTML5
- CSS3
- JavaScript
- React
- Sass
- WebPack
- NPM
- Gulp
- Weather API
