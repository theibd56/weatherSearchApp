## Installation
Install giphySearchApp with npm
```bash
  npm install weatherSearchApp
  cd weatherSearchApp
```

## Run Locally
Clone the project
```bash
  git clone https://github.com/theibd56/weatherSearchApp
```
Go to the project directory
```bash
  cd weatherSearchApp
```
Install dependencies
```bash
  npm install
```
Start the server
```bash
  npm start
```
